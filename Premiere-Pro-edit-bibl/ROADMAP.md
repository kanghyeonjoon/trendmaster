# 자동 컷편집 시스템 — 개선 아이디어 / 로드맵

> 현재 버전: `./edit.sh "영상.mp4" [--preset 보수|표준|공격]` → 무음·추임새·망설임(어/음)·더듬·false-start 제거 + 음량정리 + 자막(한줄30자/Pretendard) + 자연스러움 가드 + HTML 리포트

## 구현 완료 (2026-06-14)
- **설정 프리셋**(보수/표준/공격) + config.json — `config.py`
- **자연스러움 가드** — 컷 촘촘 구간 자동 경고 (제1원칙: 자연스러움>최대제거)
- **출력백업 + 프레임검증** — 검토 안전망
- **자막 고도화** — 한줄 30자 맥락분할, VTT/ASS(Pretendard Bold 55·그림자), CPS — `subtitle_polish.py`
- **어/음 음향 검출** — 피치평탄 지속음 검출(공격 프리셋 자동) — `acoustic_filler.py`
- **오디오 후처리** — afftdn 노이즈·deesser 치찰음 (config opt-in)
- **문맥 필러**('좀 더'는 살림) + **false-start**(비슷한 말 다시하기) 검출
- **LLM 분석** — 전사export(`transcript_export.py`) + 에이전트 하네스(리서처/기획자)가 챕터·하이라이트·숏폼·제목 수행
- **배치 처리**(`batch.sh`) + **시각 HTML 리포트**(클릭 타임코드)
- **에이전트 하네스** — 디렉터·리서처·기획자·컷편집·자막 5팀 (`video-edit-pipeline`)
- **에이전트별 도구** — 프리셋 자동추천(`analyze_video`), 9:16 숏폼 추출(`make_shorts`), 강조 키워드 자막(`emphasis_subs`), 디렉터 정량 검수 게이트, 기획자 리텐션/페이싱 맵

## A. 컷 품질 (가장 임팩트 큰 영역)

- [중간] **어/음 음향 검출** — Whisper가 어/음을 글자로 안 적는 게 근본 한계. 텍스트 대신 **음향 신호**로 잡기: 단어 사이 voiced 구간 중 피치가 평탄하게 지속되는(=망설임) 패턴을 검출. 현재 '빈틈' 방식보다 정밀.
- [쉬움] **공격성 프리셋** — 코드 수정 없이 `보수/표준/공격` 3단계 선택. `MIN_SILENCE`·`HESITATION_MIN`·필러목록을 묶어서 토글.
- [중간] **컷 클릭음 방지** — 컷 지점마다 1~2프레임 오디오 크로스페이드 자동 삽입(뚝뚝 끊기는 클릭 제거).
- [중간] **J/L 컷** — 화면은 다음 컷으로 넘어가도 앞 컷의 말꼬리는 유지(자연스러운 대화 편집 기법) 자동 적용.
- [큰작업] **문맥 기반 필러 판단** — "좀"이 '조금'(의미)인지 추임새인지, "그래서"가 접속사인지 입버릇인지 문맥(앞뒤 단어/품사)으로 구분해 선택 제거. 과제거 문제 근본 해결.
- [중간] **숨소리 제거** — 무음과 별개로 들숨/날숨 구간 감지·제거.
- [큰작업] **잘못 말한 문장(false start) 의미 검출** — 똑같은 단어 반복뿐 아니라 "비슷한 말 다시 하기"를 임베딩 유사도로 잡아 앞 시도 제거.

## B. 오디오

- [쉬움] **노이즈 제거** — ffmpeg `afftdn`/RNNoise로 배경 험·에어컨 소리 제거 단계 추가.
- [중간] **구간별 적응 음량** — 마이크 거리가 들쭉날쭉할 때 전체 단일 게인 대신 구간별 노멀라이즈.
- [쉬움] **디에서/디플로시브** — 치찰음(ㅅ,ㅊ)·파열음(ㅍ,ㅂ) 과한 부분 완화.
- [중간] **BGM 더킹** — 배경음악 있으면 말할 때 자동으로 음악 볼륨 낮춤.

## C. 자막 고도화

- [쉬움] **고유명사 용어집** — 비블 채널 자주 쓰는 단어/닉네임을 Whisper 프롬프트에 주입해 오인식("판넬빵님" 류) 감소.
- [쉬움] **숫자/맞춤법 정리** — "백만"→"100만", 띄어쓰기·문장부호 자동 정돈.
- [중간] **비블 스타일 자막 템플릿** — 채널 톤(폰트·외곽선·위치)을 프리미어 MOGRT로 만들어 자동 적용. 불러오면 바로 디자인 완성.
- [쉬움] **읽기속도(CPS) 제한 + 2줄 균형** — 너무 빨리 지나가는 자막 자동 분할, 줄바꿈 균형.
- [쉬움] **다른 포맷 출력** — VTT/ASS, 유튜브 자동자막 업로드용.
- [큰작업] **키워드 강조** — 중요 단어 자동 볼드/컬러 처리.
